# Async Query Mutation Stream Python

This example demonstrates the fundamental differences between asynchronous Query, Mutation, and Stream operations using Python's `asyncio`. It simulates a one-time data fetch (Query), a server-side data modification (Mutation), and a continuous flow of updates (Stream), highlighting their distinct behaviors and use cases.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal using Python 3.7+:
`python main.py`
